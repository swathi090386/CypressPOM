# CypressPOM

Repository for Cypress Page Object Model tests.

## Summary

This project contains Cypress end-to-end tests organized with a Page Object Model (POM) under the `cypress/e2e` directory.

## Prerequisites

- Node.js (>= 14)
- npm

## Install

```bash
npm install
```

## Run tests

- Open Cypress UI:

```bash
npx cypress open
```

- Run headless:

```bash
npx cypress run
```

## Notes

- Tests are under `cypress/e2e` and use page objects in `cypress/e2e/pages`.
- If you plan to run CI, add necessary environment variables and secrets.
